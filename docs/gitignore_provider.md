# GitIgnore Generator Provider

## Overview

The GitIgnore Generator provider generates `.gitignore` templates for different programming languages, frameworks, editors, operating systems, and other file types. This helps developers quickly set up proper ignore patterns for their projects.

## Features

- **37+ templates** covering popular languages, frameworks, and tools
- **Multi-template selection** - combine multiple templates into one file
- **Category filtering** - filter templates by category
- **Search functionality** - search templates by name or content
- **Copy to clipboard** - easily copy generated .gitignore content

## Categories

1. **Languages**: Dart, Python, Java, C++, Rust, Go, Swift, Kotlin, Ruby, PHP, CSharp
2. **Frameworks**: Flutter, Node, React, Vue, Angular, NextJS, Svelte, Laravel, Django, Spring, Rails
3. **Editors/IDEs**: VisualStudio, VSCode, IntelliJ, SublimeText, Vim, Emacs
4. **Operating Systems**: macOS, Windows, Linux
5. **Other**: Archive, Backup, Database, Image, Video, Audio, Font

## Usage

### Selecting Templates

1. Use the search field to find specific templates
2. Click on category ActionChips to filter by category
3. Click on FilterChips to select/deselect templates
4. Selected templates are highlighted

### Generating .gitignore

1. Select the desired templates
2. The generated content appears in the output section
3. Click the copy button to copy to clipboard
4. Click "Clear All" to deselect all templates
